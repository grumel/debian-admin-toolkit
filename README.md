# Debian Admin Toolkit (DAT)

Debian Admin Toolkit (DAT) is an open-source administration toolkit for Debian systems.

## Status

Current version: **0.1.0-dev**

Project status: 🚧 Under active development.

## Goals

- Modular architecture
- Plugin support
- Terminal UI (whiptail)
- Logging
- Self-update
- GitHub Actions
- Tests
- Debian 12/13 support
